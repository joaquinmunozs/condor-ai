# 📑 Sincronización a Google Sheets

Manda los leads (del diagnóstico) y clientes (del admin) a un Google Sheet automáticamente.

- `AppsScript.gs` — pegar en el Apps Script de la hoja.
- `PASOS_SHEETS.md` — guía de conexión.
- El trigger de base de datos está en `supabase/migrations/sheets_sync.sql`.
